Uma implementacao generica de backingtrack

```python
  def backtrack(v): # v é o nó sendo pesquisado
    if (promissor(v)):
      if (existe_solucao(v)):
        armazena_solucao(v)
      else:
        for filho in v:
          backtrack(filho)
  }
```
